# Inleiding

Dit document dient als documentatie voor het maken van een Startset. Het document is geschreven voor gebruikers van IMBOR, met name aan data modelleurs/architecten. Het is een aanvulling/verdieping op [deze sectie](https://docs.crow.nl/imbor/techdoc/#minimale-dataset-imbor) uit de technische documentatie van IMBOR.

# Opbouw

Dit document is opgedeeld in de secties categorieën.

1. Uitleg over wat de Startset is.
2. Wat SHACL is en hoe SHACL gebruikt kan worden om data te valideren met een Startset.
3. Hoe SHACL shapes gemaakt kunnen worden voor een Startset.
4. Hoe je valideert met Apache Jena.
